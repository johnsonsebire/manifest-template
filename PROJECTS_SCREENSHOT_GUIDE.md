## üìù Quick Screenshot Script

You can use Playwright or any screenshot tool. For Playwright, use:

```javascript
// Navigate to site
await page.goto('WEBSITE_URL');

// Take screenshot
await page.screenshot({
  path: './manifest-template/images/projects/FILENAME.png',
  type: 'png'
});
```
